üîÑ Migration Guide

To migrate existing forms to use centralized styles:

  1. Add <link rel="stylesheet" href="../assets/global.css" /> to the <head>
  2. Replace custom form styles with the classes documented here
  3. Update message divs to use class="message error" or class="message success"
  4. Remove inline styles for borders, colors, padding (now handled by classes)
  5. Use CSS variables for colors instead of hardcoded values

Before:

<div style="margin-bottom: 16px">
  <input style="border: 1px solid #e2e8f0; padding: 12px; border-radius: 8px" />
</div>

After:

<div class="form-group">
  <div class="input-wrapper">
    <input placeholder="Enter value" />
  </div>
</div>
